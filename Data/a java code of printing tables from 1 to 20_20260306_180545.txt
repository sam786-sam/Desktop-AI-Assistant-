# Introduction to Printing Tables
Printing tables is a fundamental concept in programming, and it is often used to display numerical data in a structured format. In this article, we will explore how to print tables from 1 to 20 using Java.
## What is Java?

## Java Code for Printing Tables
Here is an example of a Java code that prints tables from 1 to 20:
```java
public class PrintTables {
    public static void main(String[] args) {
        for (int i = 1; i <= 20; i++) {
            System.out.println("Table of " + i + ":");
            for (int j = 1; j <= 10; j++) {
                System.out.println(i + " x " + j + " = " + (i * j));
            }
            System.out.println();
        }
    }
}
```
### How the Code Works
1. We use a nested `for` loop to print the tables from 1 to 20.
2. The outer `for` loop iterates over the numbers from 1 to 20, and the inner `for` loop iterates over the numbers from 1 to 10.
3. Inside the inner loop, we print the multiplication of the current number `i` and the current multiplier `j`, along with the result.
4. After printing each table, we add a blank line to separate the tables.
