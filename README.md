🌐 Smart AI Desktop Assistant  
The **Smart AI Desktop Assistant** is an intelligent, voice-enabled system designed to enhance human–computer interaction through natural language commands, real-time information retrieval, and task automation. The system integrates conversational AI, web search capabilities, and system control tools to provide a seamless and efficient user experience.

---

 🚀 Features  
🔊 Voice and text-based interaction  
🧠 Decision-based query classification (General vs Real-time)  
🤖 Conversational AI using Groq API  
🌐 Real-time information retrieval using Tavily API  
⚙️ Desktop and web automation (open apps, search, play media)  
🎨 AI image generation using NVIDIA APIs  
🔉 Voice response using Edge Text-to-Speech  
💬 Chat interface for interaction history  

---

 🧩 System Architecture  
Backend Logic: Python-based control system  
Speech Recognition: Google Speech API  
AI Models: Groq API (Chatbot)  
Search Engine: Tavily API  
Automation: Selenium + PyAutoGUI + pywin32  
TTS: Edge Text-to-Speech  
Image Generation: NVIDIA API  

---

 🧠 High-Level Flow  
User Input (Voice/Text) → Speech-to-Text → Decision Model →  
→ Chatbot (General Query)  
→ Search Engine (Realtime Query)  
→ Automation Module → Output (Text + Voice)

---

 📦 Tech Stack  

| Layer | Technology Used |
